# CampusCleaner — 校园智能清扫机器人

基于 **RDK X5** 开发板与 **ROS2** 系统，搭载履带底盘及多传感器融合方案，实现自主建图、导航、动态避障与清扫作业，专为校园园区复杂环境设计，彻底解决人工清扫效率低、路线僵化及地形适应性差等痛点。

---

## 项目简介

CampusCleaner 是一款面向校园园区（道路、广场、人行道、绿化带边缘等）的无人清扫机器人。它通过 **SLAM** 实时构建环境地图，融合激光雷达、深度相机、IMU 等多传感器数据，实现厘米级定位与动态路径规划。履带底盘赋予其优越的越障能力，可轻松应对减速带、路肩、草坪等复杂地形。系统全流程自动化，支持远程监控与任务调度，显著提升清扫效率与覆盖率。

## 系统架构

### 硬件平台
| 组件            | 型号/规格                     | 作用                         |
|----------------|-----------------------------|-----------------------------|
| 主控           | RDK X5 (ARM 架构, 高算力 NPU) | 运行 ROS2 节点、算法推理       |
| 底盘           | 双电机履带式底盘              | 全地形移动、原地转向           |
| 激光雷达       | 16 线 / 单线 (可选)           | 建图、定位、障碍物检测         |
| 深度相机       | Intel RealSense D435         | 视觉避障、地面识别             |
| IMU            | 9 轴惯性测量单元              | 位姿估计、里程计辅助           |
| 清扫机构       | 滚刷 + 吸尘模块               | 垃圾收集与过滤                 |
| 电源           | 24V 磷酸铁锂电池组            | 续航 ≥ 4 小时                 |

### 软件框架
- **操作系统**：Ubuntu 22.04 LTS
- **机器人中间件**：ROS2 Humble
- **建图与定位**：Cartographer (LIDAR + IMU) + AMCL
- **路径规划**：Nav2 (A* 全局规划 + TebLocal 局部规划)
- **避障策略**：动态窗口法 (DWA) + 视觉深度图障碍物聚类
- **清扫控制**：自定义状态机 (待命 → 建图 → 规划 → 清扫 → 充电)

## 核心功能

| 功能模块         | 实现方式                                                                 |
|----------------|-------------------------------------------------------------------------|
| **自主建图**     | 手动遥控或自动遍历，生成 2D/3D 栅格地图，保存为 .pgm 及 .yaml             |
| **精准导航**     | 基于已有地图，利用 AMCL 定位，结合全局与局部规划器实时规划最优路径           |
| **动态避障**     | 激光雷达 + 深度相机实时感知动态/静态障碍物，局部路径重规划，绕行或暂停        |
| **地形适应**     | 履带底盘配合 IMU 姿态反馈，自动调整速度与驱动力，确保在斜坡、碎石路稳定行驶   |
| **清扫作业**     | 根据路径覆盖规划，自动开启/关闭清扫机构，支持边缘强化清扫模式                |
| **远程监控**     | 通过 ROS2 桥接 Web 界面，可查看实时地图、位置、电量及任务进度                |
